Sobre o Sistema

Este sistema web permite visualizar e acessar todos os dados do banco de dados da empresa de forma fácil e intuitiva.

  • Interface Web: Navegue pelos dados através do menu superior
  • APIs JSON: Acesse os dados programaticamente via endpoints REST
  • Exportação: Baixe todos os dados em formato JSON
  • Estatísticas: Visualize análises e relatórios dos dados
